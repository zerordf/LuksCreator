# LuksCreator
## This Bashs scripts is used to create a crypt volume in your system.


### Usage
```
Download https://github.com/zerordf/LuksCreator.git
cd LuksCreator
chmod +x *.sh

sudo ./LuksCreator.sh

after setup..

sudo ./LuksMount.sh
```
